# Installation

You can install Vuex ORM CRUD via NPM, Yarn.

# Dependencies

  • vuex (opens new window)
npm i vuex
  • Vuex-ORM (opens new window)
npm i @vuex-orm/core
  • axios (opens new window) (recommended)
npm i axios
