# Installation

You can install Vuex ORM CRUD via NPM, Yarn.

## Dependencies

* [vuex](https://github.com/vuejs/vuex)

```bash
npm i vuex
```

* [Vuex-ORM](https://github.com/vuex-orm/vuex-orm)

```bash
npm i @vuex-orm/core
```

* [axios](https://github.com/axios/axios) (recommended)

```bash
npm i axios
```

## Installation

```bash
npm i @wgr-sa/vuex-orm-crud
```
